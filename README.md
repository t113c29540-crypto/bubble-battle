# 🫧 泡泡大作戰 Bubble Battle

給小朋友玩的瀏覽器連線對戰遊戲——射泡泡、吃道具、90 秒內先得 10 分獲勝!

**線上玩:** https://t113c29540-crypto.github.io/bubble-battle/

## 玩法

- 射出泡泡打中對手得 1 分,先得 **10 分** 獲勝(90 秒時間到比分數,平手進入驟死戰)
- 道具:⚡ 加速 · 🛡️ 護盾(擋 2 下) · 🔱 三連發 · ⭐ 巨大泡泡

## 三種模式

| 模式 | 說明 |
|------|------|
| 🎯 單人練習 | 跟泡泡機器人(AI)對打 |
| 🏠🔑 連線對戰 | 一人「建立房間」拿 4 位代碼,朋友「加入房間」輸入代碼即可對戰(WebRTC 點對點,免伺服器) |
| 👥 同機雙人 | 兩人共用一個鍵盤對打 |

## 操作

- **電腦:** WASD 或方向鍵移動(朝移動方向發射),空白鍵 / J / 滑鼠點擊發射
- **手機:** 左半邊拖曳虛擬搖桿移動,按住右下角 🫧 連續發射
- **同機雙人:** P1 = WASD + F;P2 = 方向鍵 + Enter

## 技術

- 單一 `index.html`,無建置流程;Canvas 2D + WebAudio(音效程式合成,無外部素材)
- 連線:[PeerJS](https://peerjs.com)(WebRTC DataChannel),用官方免費雲端信令伺服器配對,遊戲資料走玩家之間點對點
- 同步模型:各自模擬自己的角色,15Hz 狀態同步;「被打中的一方」為命中權威;房主每秒校正時間與比分

## 連不上怎麼辦?

WebRTC 在少數嚴格 NAT 的網路下無法打通(無 TURN 中繼)。請兩人改連同一個 Wi-Fi,或其中一人開手機熱點分享給對方,幾乎都能解決。
